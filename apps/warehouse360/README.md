# warehouse360 — Warehouse Operations Cockpit

A high-fidelity frontend mockup for a warehouse operations cockpit, designed to provide a unified view of inbound, outbound, and inventory operations.

## 📚 Documentation

Detailed documentation is available in the `docs/` folder:

- [**Architecture Overview**](./docs/architecture.md): Overview of the frontend components and future backend plans.
- [**Local Setup Guide**](./docs/setup.md): Prerequisites and step-by-step instructions for running the mockup locally.
- [**API Reference**](./docs/api.md): Information on current static serving and future API plans.

## 🚀 Features

- **Control Tower**: Real-time operational metrics and alerts.
- **Inbound/Outbound Tracking**: Monitor shipments and schedules.
- **Inventory Visibility**: View stock levels across storage locations.
- **Production Staging**: Material flow monitoring for production lines.
- **Dispensary Management**: Weighing and dispensing raw materials.
